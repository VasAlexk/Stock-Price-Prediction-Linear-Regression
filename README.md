# Stock-Price-Prediction-Linear-Regression

# Πρόβλεψη Τιμών Μετοχών με Μηχανική Μάθηση (Amgen Inc. - AMGN)

Αυτή η εργασία υλοποιήθηκε στο πλαίσιο του μαθήματος **Στατιστικές Μέθοδοι Μηχανικής Μάθησης**. Στόχος είναι η πρόβλεψη της μηνιαίας τιμής κλεισίματος της μετοχής AMGN χρησιμοποιώντας γραμμικά και πολυωνυμικά μοντέλα παλινδρόμησης.

## Περιγραφή
Το έργο περιλαμβάνει τη συλλογή δεδομένων από το API της Alpha Vantage, την προεπεξεργασία τους με φίλτρα Gaussian και τη χρήση lag features (N=3) για την ενσωμάτωση χρονικής πληροφορίας.

### Μοντέλα που υλοποιήθηκαν:
1. **Γραμμική Παλινδρόμηση:** Βασικό μοντέλο με ανάλυση συντελεστών.
2. **Πολυωνυμική Παλινδρόμηση:** Χρήση Ridge (L2) και Lasso (L1) κανονικοποίησης.
3. **Μείωση Διάστασης:** Σύγκριση μεθόδων PCA, CFA (Correlation-based) και RFE (Recursive Feature Elimination).

## Αποτελέσματα
* **Καλύτερο Μοντέλο:** Η μέθοδος **CFA** παρουσίασε την καλύτερη γενίκευση στο validation set.
* **Αξιόπιστος Ορίζοντας:** 1 μήνας (MAE ≤ 10).
* **Πρόβλεψη Ιανουαρίου 2026:** 334.71 (βάσει μοντέλου CFA).

## Δομή Αρχείων
- `monthly_data.py`: Ανάκτηση δεδομένων από Alpha Vantage API.
- `preprocessing.py`: Καθαρισμός, φιλτράρισμα Gaussian και δημιουργία lag features.
- `A_linearModel.py`: Υλοποίηση και αξιολόγηση απλού γραμμικού μοντέλου.
- `B_polynomialModel.py`: Πολυωνυμικός μετασχηματισμός και κανονικοποίηση Lasso/Ridge.
- `C_PCA.py`, `C_CFA.py`, `C_RFE_Wrapper.py`: Μέθοδοι μείωσης διάστασης.
- `D_meCFA.py`: Τελική πρόβλεψη μελλοντικών τιμών.
